# Wiki Springfield

Bienvenido a la Wiki de Springfield, un proyecto web estático dedicado a presentar información sobre los personajes más icónicos de la serie Los Simpson.

## Descripción del proyecto

Este sitio fue desarrollado como una práctica de programación web para mostrar una interfaz sencilla, atractiva y organizada, con navegación entre páginas y contenido informativo sobre personajes de Springfield.

La página principal funciona como una entrada visual al universo del proyecto, mientras que las páginas secundarias ofrecen información detallada sobre personajes y la familia Simpson.

## Características principales

- Diseño visual atractivo con estilo propio del proyecto
- Menú de navegación entre páginas
- Secciones informativas sobre personajes
- Contenido estático y fácil de mantener
- Estructura simple basada en HTML, CSS y Bootstrap

## Tecnologías utilizadas

- HTML5 🌐
- CSS3 🎨
- Bootstrap 5 🅱️
- GitHub Pages / alojamiento estático 🚀

## Instalación

No es necesario instalar dependencias para visualizar este proyecto.

1. Clona este repositorio:
   ```bash
   git clone [URL_DEL_REPOSITORIO]
   ```
2. Abre el archivo `index.html` en tu navegador.

## Uso

Una vez abierto el sitio, puedes navegar por las distintas secciones para explorar información sobre Homer Simpson, la familia Simpson y otros personajes.
